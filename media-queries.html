Hindsight is 20/20

19th October, 2020

This was by far the hardest part of building this blog, but I learned a very valuable lesson from it: design and write your code around mobile first! I got to the point where I was happy with how my blog looked, and everything was working well. Then I remembered: "I have to make this suitable for mobile too, uh oh".

It wasn't fun, but I figured out a pretty janky way to get it to fit a mobile-width screen, and I'm damned proud that I manged that at the least, because I did become convinced there was no way I'd do it in time before the due date for this project.

From now on, if I'm building a webpage, responsiveness will be the first thing I think about, before I start wireframing, and especially before I write a single line of code.
