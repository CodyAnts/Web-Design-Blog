Learning Flexbox with an Online Game

22nd September, 2020

The in class flexbox exercise confused the living daylights out of me. The next class session we had, we were moving onto the next concept so I didn't feel there was time to ask about flexbox, otherwise I'd be behind on yet another thing. So I did the only responsible thing I could think to do. I played the flexbox minigame â€” Flexbox Froggy

As I used it, it started to click. I started to get how to use flexbox, and I was having fun doing it! Some of the levels baffled me for quite some time, and I admit I had to Google the last one. But all in all, I had a blast learning flexbox from Flexbox Froggy. Probably not the convention method of learning this particular coding technique, and I may have already forgotten most of what I learned from that exercise and likely couldn't put it to use on a real webpage. But hey, I understood in on some level, and that's a win in my books.
