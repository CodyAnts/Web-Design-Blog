Building the UXHub Homepage Structure with HTML & CSS

3rd October, 2020

With limited knowledge of HTML and CSS, me and my team began to build the home/landing page of UXHub that we had conceptualised, wireframed, and prototyped. At first we built the structure of the page with margins, padding, and all the basic CSS tools. However, we soon realised that there was no way we'd have a responsive webpage if we used those tools.

The most difficult part of HTML and CSS for me is laying out the structure of the page. I find it hard to visualise in my head what I'm actually coding. For this reason I've since began to draw out the page visually and divide each section into divs that I can then apply to the code. I also didn't think of simple methods like giving each div or container a background colour to make it far easier to see where each element is placed on the page.
